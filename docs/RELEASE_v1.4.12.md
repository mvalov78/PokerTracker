# Release v1.4.12 - Улучшения и обновления

## Дата
Февраль 2025

## Основные изменения

### Telegram-бот: ввод результата турнира
- **/result** — можно вводить только сумму выигрыша (одно число). Место сохраняется как «не указано».
- Сохранён прежний формат: `Место | Выигрыш` или `Место - Выигрыш`.
- В списке турниров и статистике место «не указано» отображается как «—» и не учитывается в «лучшем месте».

### Зависимости
- Обновлены patch/minor версии: Next.js 15.5.12, React 19.2.4, @supabase/supabase-js 2.95.3, recharts 3.7.0, zod 4.3.6 и др.
- Удалён устаревший пакет `@supabase/auth-helpers-nextjs`.
- Исправлены уязвимости (`npm audit`: 0 vulnerabilities).

### Тесты
- Исправлен мок Supabase для API-тестов (PUT турнира с результатом).
- Все 460 тестов проходят.
