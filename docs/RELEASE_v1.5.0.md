# Release v1.5.0 - Фильтрация статистики по площадке

## Дата
Февраль 2025

## Основные изменения

### Telegram-бот: фильтрация статистики по текущей площадке
- **Команда /stats** теперь показывает статистику только по турнирам текущей установленной площадки
- Если площадка установлена через `/setvenue`, статистика автоматически фильтруется по этой площадке
- В сообщении статистики отображается название текущей площадки
- Если площадка не установлена, статистика показывается по всем турнирам (как раньше)
- Если на текущей площадке нет турниров, выводится соответствующее сообщение с подсказкой

### Технические детали
- Добавлена фильтрация турниров по `venue` в методе `getStats` класса `BotCommands`
- Используется `UserSettingsService.getCurrentVenue()` для получения текущей площадки
- Добавлены тесты для проверки фильтрации статистики по площадке

### Тесты
- Добавлены 3 новых теста для команды `/stats`:
  - Статистика без фильтрации по площадке (когда площадка не установлена)
  - Статистика с фильтрацией по текущей площадке
  - Сообщение, когда нет турниров на текущей площадке
- Все тесты проходят успешно

## Пример использования

1. Установите текущую площадку: `/setvenue PokerStars Live`
2. Запросите статистику: `/stats`
3. Бот покажет статистику только по турнирам на площадке "PokerStars Live"
